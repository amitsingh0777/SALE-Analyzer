Sales Analyzer (Python Mini Project)

Sales Analyzer is a beginner-friendly Python project that demonstrates how sales data can be analyzed to extract meaningful business insights. The project uses Python libraries such as NumPy, Pandas, and Matplotlib to perform calculations, trend analysis, forecasting, and data visualization.

Project Description

This project analyzes sales data for multiple products by calculating revenue, cost, and profit. It identifies the best-performing products, evaluates monthly sales growth, and forecasts future sales using historical trends. Visual charts are generated to make the analysis easier to understand.

The project is ideal for students and beginners learning data analysis with Python.

✨ Features

-Sales dataset creation using Pandas
-Revenue, cost, and profit calculation
-Identification of:
  -Top-selling product
  -Highest revenue product
  -Most profitable product
-Monthly sales trend analysis
-Sales forecasting using average growth rate
-Data visualization with:
  -Bar chart (Revenue by Product)
  -Pie chart (Profit Contribution)
  -Line chart (Monthly Sales Trend)
